API Documentation
=================

The **stix-ramrod** APIs provide methods for updating STIX and CybOX XML
content. Listed below are the modules and packages provided by the
**stix-ramrod** library.

For examples of how make use of all of this, check out the :doc:`examples`
page.

.. note::

	The **stix-ramrod** APIs are currently under heavy development. Feel free
	to check out our `issue tracker`_ to see what we're working on!

.. _issue tracker: https://github.com/STIXProject/stix-ramrod/issues
